# Integration tests for SteadiFi

## Setting up
Step 1. Install all required packages
```bash
npm install
```

Step 2. Create a copy of `.env.example` as `.env`
```bash
cp .env.example .env
```

Step 3. Modify `.env` and fill in all required fields

## Running the tests
To run the tests simply call:
```bash
npm test
```
